# 🛠️ Jira Tools

ระบบหลังบ้านสำหรับจัดการงาน Jira และ Worklogs

## การติดตั้ง

```bash
npm install
npm run dev
```

เปิดเบราว์เซอร์ที่ [http://localhost:3000](http://localhost:3000)

## บัญชีทดสอบ

| Role  | Username | Password |
|-------|----------|----------|
| Admin | admin    | admin123 |
| User  | user     | user123  |

## ฟีเจอร์

- ✅ ระบบ Login / Logout พร้อม Role-based access
- ✅ Pastel Theme + Dark/Light Mode
- ✅ Responsive Design (PC, Tablet, Mobile)
- ✅ Loading Skeleton Animation
- ✅ Sidebar Menu แบบ Dynamic ตาม Role
- ✅ เพิ่ม Worklogs
- ✅ ตรวจสอบ Worklogs รายบุคคล
- ✅ รายงาน Worklogs ทั้งทีม
- ✅ จัดการผู้ใช้งาน (Admin)
- ✅ จัดการสิทธิ์ผู้ใช้งาน (Admin)
